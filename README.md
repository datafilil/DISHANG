⚠️ 后续迭代红线提示
禁止合并列名匹配逻辑：新浪 ETF 接口的列名（现价/成交额）与东财完全不同。未来若更换数据源，必须重新打印 list(df.columns) 验证，切勿想当然地复用旧列名。
禁止移除国家队降级标记：fund_scale_change_em 是东财独家接口，目前无公开免费平替。在接入 Tushare Pro 或自建缓存前，必须保留 ⚠️ 东财接口被封，暂缺 的显式降级，禁止让 AI 脑补国家队动向。
禁止升级 Python 版本以消除 EOL 警告：日志中的 Python 3.10 EOL 警告属于 Google 单方面的远期预告。在 CI 环境中盲目升级 Python 大版本，极易引发 akshare 底层 C 扩展编译失败，得不偿失。
禁止再次精简 Prompt 结构：完整恢复版 Prompt 中的 [系统摩擦] 进度条和元认知箴言是区分“数据播报”与“量化审计”的核心标识。任何为了节省 Token 而删减结构的行为，都会导致输出质量断崖式下跌。
